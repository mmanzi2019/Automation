# Automation Suite con Playwright 🚀

Proyecto inicial de pruebas automatizadas E2E utilizando **Playwright** y **TypeScript** en el entorno de desarrollo **Cursor**.

## 🛠️ Tecnologías utilizadas
- Playwright
- TypeScript
- Cursor IDE

## 🏃 Cómo ejecutar las pruebas
1. Instalar dependencias: `npm install`
2. Ejecutar los tests: `npx playwright test`
3. Abrir el modo interactivo: `npx playwright test --ui`
